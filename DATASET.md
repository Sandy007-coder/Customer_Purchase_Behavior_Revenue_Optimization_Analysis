# 📂 Dataset Access

The dataset used in this project is hosted externally due to size limitations on GitHub.

## 🔗 Download Dataset
Access the dataset from Google Drive:  
👉 [Customer Shopping Behavior Dataset](https://drive.google.com/drive/folders/1igL0qltICAsrDdSvdC7oJl2HZM2czQlx?usp=drive_link)

## 📥 Instructions
1. Open the link above  
2. Download the dataset files  
3. Place them inside the `data/` folder in this project  

## 📌 Dataset Details
- Contains 3,900+ customer transaction records  
- Includes customer demographics, purchase details, and behavioral data  
- Used for data cleaning, EDA, SQL analysis, and dashboard creation
