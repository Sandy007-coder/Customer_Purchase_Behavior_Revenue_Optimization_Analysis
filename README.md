📊 Customer Purchase Behavior & Revenue Optimization Analysis
📌 Overview
This project analyzes customer shopping behavior using transactional data to uncover insights that support better business decisions and revenue optimization. It demonstrates an end-to-end data analytics workflow including data preprocessing, analysis, visualization, and reporting.
📂 Dataset
Total Records: 3,900+ transactions
Features: 18 columns
Includes:
Customer demographics (Age, Gender, Location)
Purchase details (Category, Amount, Season)
Shopping behavior (Discounts, Frequency, Ratings, Shipping Type)
🛠️ Tools & Technologies
Python (Pandas, NumPy) – Data cleaning & EDA
SQL (PostgreSQL / MySQL / SQL Server) – Data querying & analysis
Power BI – Dashboard & visualization
Gamma – Presentation (PPT)
MS Word / PDF – Project report
⚙️ Project Workflow
1️⃣ Data Loading & Exploration
Loaded dataset using Pandas
Used .info() and .describe() for initial understanding
2️⃣ Data Cleaning
Handled missing values using median imputation
Standardized column names
Checked data consistency
3️⃣ Feature Engineering
Created new features like:
age_group
purchase_frequency_days
4️⃣ Exploratory Data Analysis (EDA)
Identified patterns in customer behavior
Analyzed revenue trends and product performance
5️⃣ SQL Analysis
Performed queries to extract insights:
Revenue by gender
Customer segmentation
Product performance
Shipping type comparison
Subscription vs non-subscription analysis
6️⃣ Dashboard Creation
Built an interactive dashboard in Power BI
Visualized KPIs, revenue trends, and customer segments
📊 Dashboard
The Power BI dashboard includes:
Revenue analysis by category, gender, and age group
Customer segmentation (New, Returning, Loyal)
Product performance insights
Interactive filters for dynamic exploration
📈 Key Results
Identified high-value customer segments
Analyzed revenue contribution across demographics
Found top-performing and highly discounted products
Derived insights to improve marketing and sales strategies
🚀 How to Run
Clone this repository
Open the Python files in your IDE (VS Code / Jupyter Notebook)

Bash
